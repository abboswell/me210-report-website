# ME 210 Report Website

This project is configured for GitHub Pages deployment at:

`https://abboswell.github.io/me210-report-website/`

## Local development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Push this project to the GitHub repo:
   `https://github.com/abboswell/me210-report-website`
2. On GitHub, open:
   `Settings -> Pages`
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Push to `main`.
5. GitHub Actions will build and publish the site automatically.
